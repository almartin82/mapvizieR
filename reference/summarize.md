# summarize wrapper

wrapper for summarize that preserves classes of data frames

## Usage

``` r
# S3 method for class 'mapvizieR_data'
summarize(.data, ..., .by = NULL, .groups = NULL)
```

## Arguments

- .data:

  data.frame

- ...:

  additional args

- .by:

  see
  [`summarize`](https://dplyr.tidyverse.org/reference/summarise.html)

- .groups:

  see
  [`summarize`](https://dplyr.tidyverse.org/reference/summarise.html)

## Value

data.frame
