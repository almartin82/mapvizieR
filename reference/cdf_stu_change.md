# find the RIT and NPR change across a cdf

looks at a CDF, takes first/last event by stu/subj, reports change in
RIT and NPR

## Usage

``` r
cdf_stu_change(cdf)
```

## Arguments

- cdf:

  a processed/prepped CDF, limited to the students and time period
  desired

## Value

a data frame with studentid, measurementscale, rit start/end, rit
change, npr start/end, npr change
