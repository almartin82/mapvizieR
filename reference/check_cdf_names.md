# check_cdf_names

`check_cdf_names` tests a prepped cdf's names to see if they conform

## Usage

``` r
check_cdf_names(prepped_cdf_long)
```

## Arguments

- prepped_cdf_long:

  a map CDF file, generated either by prep_cdf_long, or via processing
  done in your data warehouse

## Value

boolean, does the cdf pass?
