# grob_justifier

a great little function that baptiste wrote up on my SO queston:
http://stackoverflow.com/a/25456672/561698

## Usage

``` r
grob_justifier(x, hjust = "center", vjust = "center")
```

## Arguments

- x:

  a grob

- hjust:

  left, center, or right

- vjust:

  top, center, or bottom
