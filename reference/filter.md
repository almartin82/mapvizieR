# filter wrapper

wrapper for filter that preserves classes of data frames

## Usage

``` r
filter.mapvizieR_data(.data, ..., .by = NULL, .preserve = FALSE)
```

## Arguments

- .data:

  data.frame

- ...:

  additional args

- .by:

  see [`filter`](https://dplyr.tidyverse.org/reference/filter.html)

- .preserve:

  see [`filter`](https://dplyr.tidyverse.org/reference/filter.html)

## Value

data.frame
