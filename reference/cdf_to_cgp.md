# CDF to CGP summary

CDF to CGP summary

## Usage

``` r
cdf_to_cgp(mapvizieR_obj, cdf, grouping = "implicit_cohort", norms = 2015)
```

## Arguments

- mapvizieR_obj:

  conforming mapvizieR object

- cdf:

  conforming cdf file

- grouping:

  what column to group on. default is implicit_cohort

- norms:

  which school growth study to use. c(2012, 2015). default is 2015.

## Value

a data frame with summary start/end rit, and cgps
