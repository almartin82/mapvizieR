# Finds the centroid of a triangle

Finds the centroid of a triangle

## Usage

``` r
centroid(x1, x2, x3, y1, y2, y3)
```

## Arguments

- x1:

  first x coord

- x2:

  second x coord

- x3:

  third x coord

- y1:

  first y coord

- y2:

  second y coord

- y3:

  third y coord

## Value

a numeric vector length 2 with the xy coord of the centroid
