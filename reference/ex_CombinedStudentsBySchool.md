# example ex_CombinedStudentsBySchool file (formerly Comprehensive Data File)

Roster and demographic records, in the format provided by web-based MAP.
Data provided from NWEA, by request. (Note that this is training data,
and these students/student records \*are not real\*!)

## Usage

``` r
ex_CombinedStudentsBySchool
```

## Format

- TermName:

  TermName

- DistrictName:

  DistrictName

- SchoolName:

  SchoolName

- StudentLastName:

  StudentLastName

- StudentFirstName:

  StudentFirstName

- StudentMI:

  StudentMI

- StudentID:

  StudentID

- StudentDateOfBirth:

  StudentDateOfBirth

- StudentEthnicGroup:

  StudentEthnicGroup

- StudentGender:

  StudentGender

- Grade:

  Grade
