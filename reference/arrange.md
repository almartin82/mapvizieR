# arrange wrapper

wrapper for arrange that preserves classes of data frames

## Usage

``` r
arrange.mapvizieR_data(.data, ..., .by_group = FALSE)
```

## Arguments

- .data:

  data.frame

- ...:

  additional args

- .by_group:

  see [`arrange`](https://dplyr.tidyverse.org/reference/arrange.html)

## Value

data.frame
