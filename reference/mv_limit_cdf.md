# mv_limit_cdf

extract the cdf and limit it to target students

## Usage

``` r
mv_limit_cdf(mapvizieR_obj, studentids, measurementscale)
```

## Arguments

- mapvizieR_obj:

  a conforming mapvizieR object

- studentids:

  vector of studentids

- measurementscale:

  a MAP subject

## Value

the cdf
