# goal strand boxes

a plot that shows multiple goal scores as boxplots

## Usage

``` r
strand_boxes(mapvizieR_obj, studentids, measurementscale, fws, academic_year)
```

## Arguments

- mapvizieR_obj:

  mapvizieR object

- studentids:

  target students

- measurementscale:

  target subject

- fws:

  season

- academic_year:

  academic year

## Value

a ggplot object
