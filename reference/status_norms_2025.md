# status_norms_2025 NWEA student and school status norms data (2025 study)

Norm data published by NWEA in the 2025 MAP Growth Technical Manual.
Given a subject, grade level, season and RIT, gives a percentile rank.
Note: 2025 norms only include percentiles at 5-point intervals (5, 10,
15, ..., 95).

## Usage

``` r
status_norms_2025
```

## Format

- measurementscale:

  measurementscale

- fallwinterspring:

  fallwinterspring

- grade:

  grade

- RIT:

  RIT

- student_percentile:

  student_percentile

- school_percentile:

  school_percentile

## Source

NWEA 2025 MAP Growth Technical Manual
