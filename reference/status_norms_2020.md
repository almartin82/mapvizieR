# status_norms_2020 NWEA student and school status norms data (2020 study)

Norm data published by NWEA in the 2020 MAP Growth Normative Data
report. Given a subject, grade level, season and RIT, gives a percentile
rank.

## Usage

``` r
status_norms_2020
```

## Format

- measurementscale:

  measurementscale

- fallwinterspring:

  fallwinterspring

- grade:

  grade

- RIT:

  RIT

- student_percentile:

  student_percentile

- school_percentile:

  school_percentile

## Source

NWEA 2020 MAP Growth Normative Data
