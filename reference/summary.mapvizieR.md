# summary method for `mapvizieR` class

produces a summary for all of the objects on the main mapvizieR object.
specifically returns `mapvizieR_growth_summary` and
`mapvizieR_cdf_summary`

## Usage

``` r
# S3 method for class 'mapvizieR'
summary(object, ...)
```

## Arguments

- object:

  a `mapvizieR` object

- ...:

  other arguments to be passed to other functions (not currently
  supported)

## Value

summary stats as a `mapvizier_summary` object.
