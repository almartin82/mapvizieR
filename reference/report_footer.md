# Adds a simple footer to a report page at 1/20th the height of the existing content

Adds a simple footer to a report page at 1/20th the height of the
existing content

## Usage

``` r
report_footer(page, footer_content)
```

## Arguments

- page:

  probably the output of gridExtra::arrangeGrob

- footer_content:

  what you want the footer to say

## Value

a grob with the footer content
