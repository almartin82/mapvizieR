# process_cdf_long

`process_cdf_long` the second step in cdf processing

## Usage

``` r
process_cdf_long(prepped_cdf, norms = 2015)
```

## Arguments

- prepped_cdf:

  output of prep_cdf_long

- norms:

  norm study to used. passed through to consistent percentile

## Value

a processed cdf file
