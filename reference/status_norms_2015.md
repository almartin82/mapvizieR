# status_norms_2015 NWEA student and school status (or, attainment) norms data (2015 study)

Norm data published by NWEA: Given a subject, grade level, season and
RIT, gives a percentile rank

## Usage

``` r
status_norms_2015
```

## Format

- measurementscale:

  measurementscale

- fallwinterspring:

  fallwinterspring

- grade:

  grade

- RIT:

  RIT

- student_percentile:

  student_percentile

- school_percentile:

  school_percentile
