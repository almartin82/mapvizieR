# student_status_norms_2020_dense_extended 2020 student norm data densified and extended

Builds on status_norms_2020 for students. Two data processing steps have
been performed: 1) densification by percentile - creates entries for
intermediate percentiles 2) extending norms into HS grades where data
may be sparse

## Usage

``` r
student_status_norms_2020_dense_extended
```

## Format

- measurementscale:

  measurementscale

- fallwinterspring:

  fallwinterspring

- grade:

  grade

- RIT:

  RIT

- student_percentile:

  student_percentile

## Source

NWEA 2020 MAP Growth Normative Data
