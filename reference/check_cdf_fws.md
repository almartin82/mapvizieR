# check_cdf_fws

`check_cdf_fws` tests the fallwinterspring column of a prepped cdf to
see if it conforms

## Usage

``` r
check_cdf_fws(prepped_cdf_long)
```

## Arguments

- prepped_cdf_long:

  a map CDF file, generated either by prep_cdf_long, or via processing
  done in your data warehouse

## Value

boolean, does the cdf pass?
