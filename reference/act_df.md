# ACT linking data frame

RIT/ACT equivalencies derived from NWEA ACT linking study

## Usage

``` r
act_df
```

## Format

- school:

  I think this was put into the data to allow for rbinding with a
  different data set. Can be ignored.

- cohort:

  similar to school, can be ignored.

- grade:

  grade level season for the observation

- rit:

  RIT equivalency

- act:

  ACT sub-test equivalency

- subject:

  Mathematics or Reading

## Source

NWEA linking study
