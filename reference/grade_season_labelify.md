# grade_season_labelify

`grade_season_labelify` returns an abbreviated label ('5S') that is
useful when labelling charts

## Usage

``` r
grade_season_labelify(x)
```

## Arguments

- x:

  a cdf that has 'grade_level_season' (eg product of
  grade_level_seasonify) `grade_levelify()`

## Value

a data frame with a grade_season_labels
