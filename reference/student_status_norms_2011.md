# student_status_norms_2011 NWEA student norm data (2011 study)

Norm data published by NWEA: Given a subject, grade level, season and
RIT, gives a percentile rank

## Usage

``` r
student_status_norms_2011
```

## Format

- measurementscale:

  measurementscale

- fallwinterspring:

  fallwinterspring

- grade:

  grade

- RIT:

  RIT

- student_percentile:

  student_percentile
