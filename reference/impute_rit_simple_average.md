# Use simple averaging to impute missing rows

Use simple averaging to impute missing rows

## Usage

``` r
impute_rit_simple_average(cdf, interpolate_only = TRUE)
```

## Arguments

- cdf:

  a CDF data frae

- interpolate_only:

  should the scaffold return ALL seasons, ever, or only ones in between
  the student's first/last test?

## Value

a CDF data frame with imputed rows
