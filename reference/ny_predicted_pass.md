# New York predicted pass rate

Predicted NYS proficiency from NWEA linking study

## Usage

``` r
ny_predicted_pass
```

## Format

- grade:

  student's grade level

- measurementscale:

  NWEA measurementscale

- fallwinterspring:

  season

- percentile:

  percentile, 2015 norms

- pass_probability:

  probability of achieving a level 3 or level 4

## Source

NWEA New York linking study
