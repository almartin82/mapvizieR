# mapvizieR Color Palettes

Standard color palettes used throughout mapvizieR visualizations.

## Usage

``` r
mapvizier_quartile_colors()

mapvizier_growth_colors()

mapvizier_kipp_colors()
```

## Value

A character vector of color hex codes

## Examples

``` r
mapvizier_quartile_colors()
#>         1         2         3         4 
#> "#f3716b" "#79ac41" "#1ebdc2" "#a57eb8" 
mapvizier_growth_colors()
#>      Negative      Positive       Typical College Ready 
#>     "#FF0000"     "#CCFF00"     "#0066FF"     "#CC00FF" 
```
