# example ex_CombinedAssessmentResults_pre_2015 file (formerly Comprehensive Data File)

MAP assessment results (\*\*pre-2015\*\*), in the format provided by
web-based MAP. Data provided from NWEA, by request. (Note that this is
training data, and these students/student records \*are not real\*!)

## Usage

``` r
ex_CombinedAssessmentResults_pre_2015
```
