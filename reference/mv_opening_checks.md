# mv_opening_checks

common beginning checks when building a plot. DRY, right?

## Usage

``` r
mv_opening_checks(mapvizieR_obj, studentids, min_stu = 1)
```

## Arguments

- mapvizieR_obj:

  a valid mapvizieR object.

- studentids:

  vector of studentids to run for this plot

- min_stu:

  minimum number of students for this plot. default is 1.
